<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<EXECUTION>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

To pull from github:
1. Use terminal.
2. "git pull origin main"

To run frontend:
1. Use the terminal, make your way to /Medipal/frontend
2. run "npm install" (If you have missing dependencies).
3. run "npm run dev"
